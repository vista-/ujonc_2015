There is this awesome old game called Master of Adventures 5, that you really want to play, but it requires a serial number. You would buy the game if you could, but the publisher has long disappeared, and there is no way to actually buy the game.

The only way is to somehow generate a serieal number based on the game binary. To make this easier, a friend of yours extracted the actual serial number checking code into a separate binary file that can be run on Linux. This file waits for a string on its standard input, and print ':)' if the serial check was successful. If it was not, it prints ':('.
